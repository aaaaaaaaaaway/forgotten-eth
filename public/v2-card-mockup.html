What changed for D3:
1. The left rail takes the token's brand color (orange/yellow/blue/green) instead of the default accent teal — instant visual classification at a glance.
2. A small uppercase token chip sits inline next to the protocol name, outlined in the token color. Same vocabulary as the hero sub-counter labels.
3. The amount stays gold for the number (matching ETH cards) and gets a small dimmed DAI/USDC/USDT/WBTC suffix in place of "ETH". Same font, same size, just relabelled.
4. The Release button picks up the token color too, reinforcing what's being claimed at the moment of action.
5. Hover state: border and box-shadow shift to the token color, again — instead of the default accent.

The ETH card at the top is the baseline reference — completely unchanged from production. Only non-ETH cards get the chip + per-token coloring. A user with mixed ETH + DAI claims sees the same flat list, with the v2 entries gently color-coded.
